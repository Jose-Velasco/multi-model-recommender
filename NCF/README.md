# Prerequisites

1. Python 3.7+
2. pip 
3. Install required libraries: pip install tensorflow numpy pandas matplotlib tqdm

# Usage

1. Prepare your data: Ensure train-1.txt exists in the project folder.
2. Run the script: python mlp.py/neumfv2.py
3. Training: The model trains for a default of 5 Epochs.

# Output

1. A plot window will appear showing the loss and NDCG curves.
2. Results are saved to output.txt.
